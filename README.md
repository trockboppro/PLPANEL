# PLPANEL Dashboard

![PLPANEL Logo]()

Plpanel is a sleek and efficient dashboard designed for managing your resources with ease. While Discord login isn’t implemented yet, you can still enjoy a seamless experience managing your system.

### Installation

To install and start the Oversee dashboard, run the following commands:

```bash
git clone https://github.com/trockboppro/PLPANEL.git && cd Oversee && npm install && npm run seed && npm run createUser && node .
```
